What are the five elements of a trigger's spec?
Name, type, firing granularity, correlation names, conditional specification

How do you invoke a trigger?
when situation described by type occurs - run the firing event.

WHen do you use the keyword DECLARE in a trigger?
when declaring variables, omitted if there are none.

What is the error that occurs when your trigger is trying
to change an already changing item?
mutating trigger error

What is the name of the table in the data dictionary that
stores info about your triggers?
user_triggers

How do you remove a trigger once it's been created?
DROP TRIGGER trigger_name;
