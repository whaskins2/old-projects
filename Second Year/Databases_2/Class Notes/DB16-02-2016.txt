A function with no parameters has parenthesis () in the spec?
FALSE

What is the naming convention for parameters?
in_parameter_name

How do you execute a function with no parameters?
in a procedure on the right hand side of an assignment statement eg:
variable_name := function_name;

How do you execute a function with parameters?
variable_name := function_name(argument);

How do you remove a function once its been created?
DROP FUNCTION function_name

CREATE OR REPLACE FUNCTION username_funct
(ic_firstname VARCHAR2, ic_surname VARCHAR2) RETURN VARCHAR2 IS
vc_username VARCHAR2(10);

BEGIN
vc_username := CONCAT(SUBSTR(in_firstname,1,2), SUBSTR(in_surname,1,5))

RETURN vc_username;

END username_funct;
/

CREATE OR REPLACE PROCEDURE eg_func
(ic_firstname VARCHAR2, ic_surname VARCHAR2) IS

vc_username VARCHAR2(10);

BEGIN

vc_username := username_funct(ic_firstname, ic_surname);
